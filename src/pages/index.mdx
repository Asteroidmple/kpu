# 政治·经济·文化·历史解析

知乎的创作生态被破坏的越来越严重，曾经我以为能对抗伪史论维护历史真相和学术正义，后来我以为靠MLM主义能让我正确的理解社会，可我现在发现这些都没什么意义，这个平台充斥着两个极端，让我越来越不想参与政治议题，他们的发言极端，冷血，残酷，又偶尔显露出一丝微不足道的正义——就是伪装在爱国主义之下的褐色瘟疫，即使是一些我曾经很尊敬的人，也有这种现象。2019年我和神友决裂，2021年我和兔友决裂，如今我也不知道谁是对的，不过越来越明显的感受到，他们都是错的，至少很多地方并不能称之为正确。
我非常讨厌很多人把国家意志加诸自身，他们的行为就是“只恨我自己不是帝国主义”，实际上不代表任何先进性，当然也不代表我自己。国家意志往往是一台冷酷到无情的机器，它在历史前进的进程中充当了车轮和碾压机的角色，把很多无辜的生灵碾碎，它没有人性，更不神圣。所以我更想保留一份人性的善在自己的内心，这种善植根于我的内心和所受的教育，不为外界所改变。
